# angelicxs-RecycleJob

A simple and optimized recycling job script for FiveM servers. This resource allows players to collect recyclable materials, process them, and earn rewards through an immersive recycling system.

* ✨ Features
* ♻️ Full recycle job system
* 📦 Collect and process recyclable materials
* 💰 Reward and payment system
* 🧍 Optimized NPC and interaction zones
* ⚡ Lightweight and performance friendly
* 🔧 Easy configuration
* 🧩 Framework support ready
* 🎮 Roleplay server friendly

### 📦 Dependencies

Before installing this script, make sure you have the following dependencies installed on your server:

* [ox_lib](https://github.com/overextended/ox_lib)
* [ox_target](https://github.com/overextended/ox_target)
* [ox_inventory](https://github.com/overextended/ox_inventory)

### Optional:

* QBCore / ESX compatible setups
---
## 📥 Installation

1. Download or clone the repository:

```bash
git clone https://github.com/PehesaraNimnad/angelicxs-RecycleJob.git
```
2. Place the folder inside your resources directory.
3. Add the resource to your server.cfg:
   * ensure ox_lib
   * ensure ox_target
   * ensure ox_inventory
   * ensure angelicxs-RecycleJob
---
# 🚀 Usage

* Go to the recycling center
* Collect recyclable materials
* Process the materials
* Receive rewards and payments
